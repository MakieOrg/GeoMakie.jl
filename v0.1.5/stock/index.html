Stock data

GeoMakie offers coastlines and an image of the Earth (both from Natural Earth) as pre-loaded datasets, which you can plot using coastlines or earth:

GeoMakie.coastlines — Function
coastlines(; kwargs...)

Plot coastlines along the world.

GeoMakie.earth — Function
earth(; kwargs...)

Plots an image of the Earth in the given projection.
